# 版本管理

这里汇总了 WebRTC Player 的历史版本发布记录与关键演进路径。

## 版本演进概览

- **1.x 阶段**：完成项目基建，形成拉流 + 推流双核心能力，并持续优化文档与工程结构。
- **2.0 阶段**：围绕插件体系和稳定性做重构，引入重连机制，提升生产可用性。
- **2.1 阶段**：继续增强插件扩展上限，补齐关键 Hook 与事件链路，修复重连边界问题。

## 阅读建议

- 如果你是新接入用户，建议从 `v2.1.0` 开始阅读，了解当前推荐能力。
- 如果你是历史版本升级用户，建议先阅读目标版本的“升级建议”和“兼容性影响”。
