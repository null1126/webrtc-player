# v2.0.0

> 以稳定性与中长期扩展为目标的大版本升级。

## 版本亮点

- 重构插件体系，提升复杂场景下的可维护性。
- 新增重连机制，提升弱网与异常场景下的可用性。

## 详细更新

### 插件系统重构

- 优化插件内部组织方式与执行链路。
- 提升插件协同能力，减少功能叠加时的冲突风险。
- 为后续更高扩展上限（如更多 Hook 与更细粒度事件）预留空间。

### 重连机制新增

- 在连接中断后支持自动重连流程。
- 改善网络波动场景下的恢复能力。
- 降低用户侧手动干预频率，提升连续播放/推流体验。

## 影响范围

- 主要影响插件相关接入方式与连接恢复策略。
- 历史项目若深度依赖旧插件行为，建议先在测试环境验证。

## 升级建议

- 升级前建议梳理现有插件职责与执行顺序。
- 升级后建议重点验证网络抖动、断网恢复、长连接稳定性等场景。
