# v1.0.0

> 首个可用版本，确立 WebRTC Player 的基础架构与核心能力边界。

## 版本亮点

- 完成项目基建与工程化骨架。
- 提供可直接使用的拉流器（Player）能力。
- 提供可直接使用的推流器（Publisher）能力。

## 详细更新

### 工程基础

- 建立统一的 TypeScript 项目结构与构建流程。
- 约定核心模块边界，为后续能力扩展预留接口。
- 初始化文档体系与基础示例，降低首次接入成本。

### 拉流器能力

- 支持通过 WebRTC 拉取远端媒体流并播放。
- 提供基础生命周期控制（创建、启动、停止、销毁）。
- 暴露关键状态与错误信息，便于业务层处理异常。

### 推流器能力

- 支持采集本地摄像头、麦克风并推送到媒体服务器。
- 提供基础的会话与连接管理能力。
- 具备最小可用的参数化配置，满足常见接入场景。

## 适用场景

- 快速验证 WebRTC 拉流/推流闭环。
- 构建轻量级直播、实时互动原型。

## 升级提示

- 该版本为基线版本，后续版本会在插件化、稳定性与扩展性方面持续增强。
